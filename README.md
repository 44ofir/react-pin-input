# webpack-react-lib-starter

**React component for PIN/MPIN like input**

[![Build Status](https://travis-ci.org/arunghosh/react-pin-input.svg?branch=master)](https://travis-ci.org/arunghosh/react-pin-input)


### For developers

**To make new build**
```
npm run build
```

**To run dev server**
```
npm run dev
```

**To run test**
```
npm run test
```
